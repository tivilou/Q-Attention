# CLAUDE.md

## Project Goal

This repository is for an original quantum machine learning attention-steering project. Future source code should be written from scratch.

## Working Rules

- Do not import or copy external project source code into this repository.
- Keep the public repository focused on original implementation and design notes.
- Add citations in papers and manuscripts when they become relevant, but do not vendor third-party source here.
- Keep default prototypes lightweight and torch-only unless a quantum framework is explicitly needed.

## Research Themes

1. Quantum-kernel adaptive routing.
2. Quantum-inspired spectral projector filtering.
3. Quantum separability-guided head and layer selection.

## First Implementation Target

Start with a minimal quantum-kernel routing module under `src/q_attention/`.
